Welcome to stix2-generator's documentation!
===========================================

The STIX 2 Generator is a software tool for generating random STIX content for
prototyping and testing. It uses a simple, sentence-like syntax for expressing
what STIX content to generate. This tool is provided in two forms: as a Jupyter
notebook and as a commandline tool.
